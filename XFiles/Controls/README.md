# Controls/

XAML UserControls for the file browser UI:

- `ColumnListView.xaml(.cs)` — one Miller column (Parent/Current), gamepad-focusable list
  with custom ControlTemplate (see docs/UI-THEMING.md). Introduced in Phase 3/4.
- `PreviewPane.xaml(.cs)` — right-most column, switches between folder listing / text /
  image / archive-listing / unsupported templates via a DataTemplateSelector. Introduced
  in Phase 5.

Nothing implemented yet — see docs/ROADMAP.md.
