---Resource Redirect (Windhawk)---

Windhawk's Resource Redirect allows users to modify Windows 11 resources without directly altering system files. This is achieved by redirecting the system to load resources from a user-specified folder instead of the default location. 

Download and Install Windhawk: Download the latest version of Windhawk from the windhawk.net. 
Install the Resource Redirect mod: Within Windhawk, locate and install the "Resource Redirect" mod. 
Create or locate your resource folder: Prepare a folder containing the resources you want to redirect. This could be a theme, icons, or any other Windows resource. 
Open Windhawk's Resource Redirect settings: Go to the "Resource Redirect" mod's settings within Windhawk. 
Set the redirect path: In the settings, provide the path to the folder containing your resources. Remove any quote marks from the path. 
Enable "Redirect all loaded resources": Enable this option to ensure that all resources are loaded from your specified folder. 
Apply changes: After configuring the settings, apply them within Windhawk. 
Test the redirection: Check to see if the resources are being loaded from your new location. 

Sometimes you may need to rebuild or refresh your icon Cache for notices to take effect. - https://www.elevenforum.com/t/rebuild-icon-cache-in-windows-11.2049/

---How to change the sidebar icons---

***Winaero Tweaker*** - https://winaero.com/winaero-tweaker/

File Explorer > Navigation Pane - Default Items (Clear all but This PC)
File Explorer > Navigation Pane - Custom Items.  Add as many as you like.  Customize it to your needs.

Plus some extra registry modifications are recommended to remove unused shortcuts such as Home and Gallery.
